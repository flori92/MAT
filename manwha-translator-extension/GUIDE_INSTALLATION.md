# 📖 Guide d'Installation - Manwha Translator

## Méthode 1 : Installation Manuelle (Recommandée)

### Étape 1 : Télécharger l'extension
1. Téléchargez le fichier `manwha-translator-v1.0.0.zip`
2. Décompressez-le dans un dossier de votre choix

### Étape 2 : Chrome / Edge / Brave

1. **Ouvrez** votre navigateur
2. **Tapez** dans la barre d'adresse :
   - Chrome : `chrome://extensions/`
   - Edge : `edge://extensions/`
   - Brave : `brave://extensions/`
3. **Activez** le "Mode développeur" (bouton en haut à droite)
4. **Cliquez** sur "Charger l'extension non empaquetée"
5. **Sélectionnez** le dossier décompressé de l'extension

### Étape 3 : Firefox

Le support Firefox est actuellement expérimental. La cible principale reste `Chrome`, `Edge` et `Brave`.

## 🎯 Utilisation Rapide

### Traduire une page entière
1. Allez sur un site de manwha (ex: Asura Scans)
2. Ouvrez un chapitre
3. **Cliquez** sur l'icône 🌸 dans la barre d'outils
4. **Cliquez** sur "Traduire cette page"
5. Attendez quelques secondes...
6. ✅ Les bulles de dialogue sont maintenant en français !

### Traduire une image spécifique
1. **Faites un clic droit** sur l'image à traduire
2. **Sélectionnez** "🌸 Traduire cette image"

### Mode automatique
1. Ouvrez l'extension 🌸
2. **Activez** "Traduction auto"
3. Les nouvelles images seront traduites automatiquement

## ⚙️ Configuration

### Modes d'affichage

| Mode | Description | Quand l'utiliser |
|------|-------------|------------------|
| **Superposé** | Traduction visible au survol | Lecture normale |
| **Remplacer** | Remplace définitivement le texte | Sauvegarde |
| **Bulles FR** | Crée de nouvelles bulles | Meilleure lisibilité |

## 🐛 Résolution des problèmes

### "Aucune image détectée"
→ Rafraîchissez la page et réessayez

### "La traduction ne s'affiche pas"
→ Vérifiez votre connexion internet
→ Certains sites peuvent empêcher le chargement OCR ou l'accès direct aux images

### "Le texte est mal positionné"
→ Changez le mode d'affichage

### "C'est trop lent"
→ Passez en mode "Rapide"

## 💡 Astuces

- **Épinglez** l'extension pour un accès rapide
- Utilisez le **mode Bulles FR** pour une meilleure expérience
- Activez **"Conserver l'original"** pour comparer
- La traduction est **mise en cache** - plus rapide la 2ème fois !
